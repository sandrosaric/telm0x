# Contributing

This is a craft project. PRs are welcome, but we value silence over features.

## Principles

- **Silence over noise.** If it doesn't make the SDK smaller, faster, or more reliable, it probably doesn't belong here.
- **No breaking changes.** Once shipped, API signatures stay stable.
- **Fire-and-forget.** Telemetry must never block or crash the host application.

## Submitting PRs

1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Submit a PR

We'll review it. If it aligns with the project's philosophy, we'll merge it.
